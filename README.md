# Gym-Application
Python project built using FastApi and SQLite

🏋️ Gym Fitness App

A modern Gym & Fitness Management Application built using Python and FastAPI as the backend.
This application helps users manage workouts, track calories, and explore fitness-related information through a clean and simple interface.

🚀 Features
🏠 Home Dashboard
🍎 Calories Tracking Section
💪 Workout Management Section
⚡ FastAPI Backend
📊 REST API Support
🔥 Lightweight & Fast Performance
🗂 Organized Project Structure
📌 Sections
🏠 Home Section

The Home section acts as the main dashboard of the application.

Features:
Welcome page for users
Navigation to all sections
Fitness overview
Simple and responsive UI
🍎 Calories Section

The Calories section helps users manage and track their daily calorie intake.

Features:
Track calories consumed
View calorie information
Daily calorie monitoring
Health-focused fitness tracking
💪 Workout Section

The Workout section allows users to explore and manage exercises and workout routines.

Features:
Workout categories
Exercise details
Daily workout planning
Fitness activity management
